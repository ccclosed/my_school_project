# Rust Kernel (i686)

Простое ядро операционной системы на Rust для архитектуры i686 (32-bit x86).

## Возможности

### Базовые компоненты
- **GDT/IDT** - сегментация и обработка прерываний
- **Управление памятью** - собственный аллокатор кучи (2 MB), пейджинг с identity mapping
- **Многозадачность** - round-robin планировщик на 8 задач с переключением контекста по таймеру
- **VGA драйвер** - текстовый режим 80x25 с поддержкой scrollback буфера (500 строк)

### Сетевой стек
- **RTL8139** - драйвер сетевой карты Realtek
- **Ethernet** - парсинг и создание фреймов
- **ARP** - разрешение MAC-адресов
- **IPv4** - базовая поддержка IP
- **UDP** - транспортный протокол
- **DHCP клиент** - автоматическое получение IP-адреса

### Файловая система
- **RamFS** - простая файловая система в памяти
- Операции: создание, чтение, запись, удаление файлов

### Shell
Интерактивная командная оболочка с историей команд (Up/Down):
- `help` - список команд
- `ls` - список файлов
- `touch/cat/write/rm` - работа с файлами
- `edit <file>` - текстовый редактор (Esc=выход, Ctrl+S=сохранить)
- `calc <expr>` - калькулятор (поддержка +, -, *, /, %, скобки)
- `mem` - статистика кучи
- `cpuinfo` - информация о процессоре
- `netinfo` - статус сетевой карты
- `dhcp` - запрос IP через DHCP
- `spawn <name>` - создать задачу
- `ps` - список задач
- `uptime` - время работы системы
- `env` - системная информация
- `shutdown` - выключение

## Сборка

### Требования
- Rust nightly (GNU toolchain для Windows)
- QEMU (для запуска)
- WSL с GRUB (для создания ISO, опционально)

### Компиляция
```bash
.\build.ps1
```

### Запуск в QEMU
```bash
.\run-qemu.cmd
```

Или с ISO:
```bash
.\mkiso.ps1
.\run-qemu.cmd -iso
```

## Архитектура

### Память
- **Heap**: 2 MB, linked list allocator
- **Stack**: 64 KB на задачу
- **Paging**: Identity mapping 0-64MB

### Прерывания
- IRQ0 (Timer) - переключение задач
- IRQ1 (Keyboard) - ввод с клавиатуры
- IRQ12 (Mouse) - мышь (частично)
- Exceptions - Page Fault, GPF, Double Fault и т.д.

### Сеть
- Поддержка RTL8139 (QEMU: `-net nic,model=rtl8139`)
- TX: 4 дескриптора с per-descriptor буферами
- RX: кольцевой буфер 8KB

## Известные ограничения
- Только 32-bit (i686)
- Нет userspace (всё в ring 0)
- Нет TCP (только UDP)
- Нет файловых систем на диске (только RamFS)
- Scheduler без приоритетов
